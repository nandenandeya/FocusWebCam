SOURCE IMAGES FOLDER
=====================
Tempatkan file gambar kamu di sini:

  bg.png      — Background utama (landing page & app page)
                Ukuran ideal: 1440x900px atau lebih besar
                Format: PNG / JPG

  logo.png    — Logo FocusWebCam (pojok kiri landing page)
                Ukuran ideal: max height 64px, transparent background
                Format: PNG (dengan transparansi)

  bg2.png     — (opsional) Background alternatif

Cara mengganti gambar:
  Cukup replace file dengan nama yang sama.
  Tidak perlu mengubah kode apapun.
